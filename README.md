# 英単語帳おすすめ診断アプリ

ユーザーの学習スタイルの好みと現在の英語レベルに基づき、最適な英単語帳を提案するWebアプリケーションです。

##  概要

このアプリケーションは、ユーザーが重視したい学習要素の優先順位を入力することで、データベースに登録された主要な英単語帳の中から、最も相性の良い一冊をマッチングします。


##　アプリケーションの実行

以下のコマンドでサーバーを立ち上げます。
```Bash
python app.py
```
指定されたURLを開くことでアプリケーションを利用することができます。
